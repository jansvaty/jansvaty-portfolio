# Jan Svaty Portfolio

Personal portfolio website for Jan Svaty, built with HTML, CSS, and JavaScript.

## Local preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## Deployment

This site is deployed with GitHub Pages using the workflow in `.github/workflows/pages.yml`.

## Live site

https://jansvaty.com
